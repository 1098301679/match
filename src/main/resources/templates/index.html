智慧校园应用系统 的页面采用流行的 Bootstrap 开发, 所有页面整体布局使用的 Bootstrap 提供的流式的栅格系统. 页面 JS 特效使用了少量的 jQuery.

系统分为前台展示页面及后端管理页面,后端采用 Spring Boot 与 Mybatis 开发, 模板引擎选用 Thymeleaf，前台数据展示主要使用 ECharts、Vue 与 Axios 等技术实现.

Axios

Axios 是一个基于 promise 的 HTTP 库，可以在浏览器中轻松使用, 支持所有请求类型, 也可一次发送多个并发请求, 轻量且易用.
